# Calculator

Simple accessible calculator (HTML/CSS/JS).

Features:
- Basic arithmetic: +, -, *, /
- Decimal numbers and parentheses
- Keyboard support (digits, + - * / ( ) . Enter for equals, Backspace, Escape)
- Simple input validation to allow only arithmetic characters

How to run:
Open `index.html` in your browser (double-click or serve with a static file server).

Notes:
- This is a static client-side app. No build step required.
- Error states display `Ошибка` and reset the input.
