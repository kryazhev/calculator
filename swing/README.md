# Java Calculator - Desktop приложение

Простой и функциональный калькулятор на Java с использованием Swing.

## Функции

- Базовые арифметические операции (+, -, ×, ÷)
- Очистка всего (C)
- Очистка текущего ввода (CE)
- Удаление последней цифры (⌫)
- Изменение знака (±)
- Работа с десятичными числами

## Компиляция и запуск

### Компиляция:
```bash
javac swing/Calculator.java
```

### Запуск:
```bash
java -cp . swing.Calculator
```

## Требования

- Java JDK 8 или выше
- Не требует дополнительных библиотек (использует стандартный Swing)

## Особенности

- Современный и интуитивный интерфейс
- Цветовая кодировка кнопок для лучшей читаемости
- Обработка ошибок (деление на ноль)
- Поддержка клавиатуры и мыши
